# avik/crate

Dependency injection container for the Avik framework.

## Features
- Interface to implementation binding
- Singleton support
- Automatic constructor injection
- Parameter overrides

## Design Rules
- Explicit bindings
- No magic scanning
- Minimal API surface

## Stability
Frozen after v1.0.0

## Requirements
- PHP 8.1+
- avik/seed ^1.0
